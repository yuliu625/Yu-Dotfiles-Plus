# Path Toolkit
## 概述
一个基于`pathlib`，用于简化路径处理的工具集。

在软件开发中，路径处理是一项基础但繁琐的任务。这个仓库收集和整理我在各种任务中构造的一系列实用的、可复用的路径处理方法。

这里的工具都基于Python标准库`pathlib`构建，它们可以作为独立的脚本直接集成到具体项目中。


## 说明
这个项目的构造初衷是: 构建一个比现有工具更强大的路径处理库。然而，在实践过程中，我发现:
- `pathlib`已经足够强大，能够满足大多数场景的需求。
- 路径处理的需求往往高度具体，很难预先设计一个通用性极强的工具。

因此，我将项目的重心调整为: 收集各种可能会被重复使用的路径处理工具函数。这些工具更像是实用的代码片段，而不是一个庞大、复杂的框架。
